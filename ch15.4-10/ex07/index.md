### .png 画像や .jpeg 画像と比べた際の SVG のメリット・デメリットを調査しなさい。

### SVG 画像のメリット

- 拡大・縮小が自在
  SVG はベクター形式の画像フォーマットであるため、解像度に依存せずに拡大・縮小が可能である。これにより、異なる画面サイズや解像度に対応しやすくなる。
- ファイルサイズが小さい
  SVG はテキストベースのフォーマットであり、複雑な画像でも比較的ファイルサイズが小さくなることが多い。これにより、ウェブページの読み込み速度が向上する。
- スタイリングとアニメーションが可能
  SVG は CSS や JavaScript を使用してスタイリングやアニメーションを適用できるため、動的なグラフィックスを作成するのに適している。

### SVG 画像のデメリット

- 複雑な画像には不向き
  SVG はベクター形式であるため、写真のような複雑な画像には不向きであり、JPEG や PNG の方が適している場合がある。
- ブラウザの互換性
  一部の古いブラウザでは SVG のサポートが不完全であるため、互換性の問題が発生する可能性がある。
- 学習コスト
  SVG の構造や属性を理解するためには一定の学習コストが必要になる。
